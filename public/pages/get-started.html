Get started

1. Install Owl framework

1.1 Install from generated sources


git clone https://github.com/owl-framework/owl
cd owl/ext
sudo ./install

2. Create a project

First you need choose, what an application you will build.

Vagrant Boxes

Box #1

  • Ubuntu 14.04 Server (Trusty) x64
  • PHP 5.6
  • Zephir dev-master
  • Owl dev-master

git clone https://github.com/owl-framework/vagrant ./owl-vagrant
cd owl-vagrant
nano config.yaml //edit you settings
vagrant up
